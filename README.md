# Cordova

### Shortcut Xcode

> Cmd + Maj + K = Cache Clear

> Cmd + Q = Quit simulator

> Cmd + R = Run simulator


Faire un cordova build (ou prepare) pour que cordoba interprète les plugins (splashscreen)


### Plugins

- ` cordova plugin search ` ...
- ` cordova plugin ls `  (liste des plugins utilisés)
